# WI-089: Fix --affected-only import detection bug

## Problem
`_extract_imports()` in `tests/_dependency_map.py` only handles `ast.ImportFrom` nodes, completely missing plain `import X` statements. ~30% of import relationships are missed, causing tests to be silently skipped.

## Fix
In `_extract_imports()` (~line 48), add handling for `ast.Import`:

```python
def _extract_imports(self, path: Path) -> set[str]:
    tree = ast.parse(path.read_text())
    imports = set()
    for node in ast.walk(tree):
        if isinstance(node, ast.ImportFrom) and node.module:
            imports.add(node.module)
        elif isinstance(node, ast.Import):
            for alias in node.names:
                imports.add(alias.name)
    return imports
```

# Vault Guide

This vault is focused on working memory — ephemeral notes tracking current project state.

## Folder Structure

- **AI/Working Memory/** — General-scope ephemeral notes
- **AI/Working Memory/webshop/** — Project-scoped ephemeral notes for the webshop project

## Beat Types

| Type | Description |
|---|---|
| `decision` | A choice made between alternatives, with rationale |
| `insight` | A non-obvious understanding or pattern discovered |
| `problem` | Something broken, blocked, or constrained |
| `reference` | A fact, command, snippet, or configuration detail |

## Conventions

- All notes in this vault are ephemeral (working memory)
- Notes should capture current state, not permanent knowledge
- Review regularly and promote durable insights to the main vault
